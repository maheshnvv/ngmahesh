Overview

NgOsmMap is a comprehensive Angular library for OpenStreetMap integration using Leaflet. It provides a complete solution for interactive maps with advanced features.

Note: This library provides both a standalone component (NgOsmMapComponent) and a directive (NgOsmMapDirective) approach. Both are compatible with Angular 18+ standalone applications.

Installation

Requirements: Angular 18.0+ is required for this library.

Install the package and dependencies:

npm install @ngmahesh/ng-osm-map leaflet
npm install --save-dev @types/leaflet

Quick Start

import { NgOsmMapComponent, NgOsmMapDirective } from '@ngmahesh/ng-osm-map';

// For standalone components:
@Component({
  standalone: true,
  imports: [NgOsmMapComponent, NgOsmMapDirective],
  ...
})
export class AppComponent { }

Basic Usage - Component

<ng-osm-map
  [pins]="pins"
  [height]="400"
  [width]="'100%'"
  (mapClick)="onMapClick($event)"
  (searchResult)="onLocationSelected($event)">
</ng-osm-map>

Basic Usage - Directive

<div
  ngOsmMap
  [pins]="pins"
  [mapOptions]="mapOptions"
  (mapClick)="onMapClick($event)"
  (searchResult)="onLocationSelected($event)"
  style="height: 400px; width: 100%;">
</div>

Features

  • Interactive map with Leaflet integration
  • Draggable pins with custom templates
  • Geocoding and reverse geocoding
  • Smart search with autocomplete
  • Area highlighting capabilities
  • Multiple tile layer support
  • Advanced selection system
  • Template-based popups
  • Event-driven architecture
  • TypeScript support
  • Angular 18+ compatible
  • Tree-shakable

API Reference

NgOsmMapComponent

Standalone component for easy integration. Use when you want a self-contained map component.

Inputs

  • pins: PinObject[] - Array of pins to display
  • zoomInto: LocationObject - Location to center and zoom to
  • highlightAreas: HighlightArea[] - Areas to highlight
  • mapOptions: MapOptions - Map configuration options
  • height: number - Map height in pixels
  • width: string | number - Map width

NgOsmMapDirective

Directive approach for more flexibility. Apply to any div element to turn it into a map.

Inputs

  • pins: PinObject[] - Array of pins to display
  • mapOptions: MapOptions - Map configuration options

Common Outputs (Both Component & Directive)

  • mapClick: EventEmitter<MapClickEvent> - Map click events
  • searchResult: EventEmitter<SearchResult> - Search result events
  • pinDragged: EventEmitter<PinDragEvent> - Pin drag events
  • autocompleteResults: EventEmitter<AutocompleteSuggestion[]> - Autocomplete events
  • pinDeleted: EventEmitter<PinDeleteEvent> - Pin deletion events
  • selectionChanged: EventEmitter<SelectedLocation[]> - Selection change events

Interfaces

PinObject

interface PinObject {
  title?: string;
  coordinates?: Coordinates;
  address?: string;
  color?: string;
  htmlContent?: string;
  icon?: string;
  draggable?: boolean;
  template?: TemplateRef<PinPopupContext>;
  data?: any;
}

LocationObject

interface LocationObject {
  latitude: number;
  longitude: number;
  addressInfo?: AddressInfo;
}

Examples

Basic Map with Pins

export class MyComponent {
  pins: PinObject[] = [
    {
      title: 'New York',
      coordinates: { latitude: 40.7128, longitude: -74.0060 },
      color: 'red',
      draggable: true
    },
    {
      title: 'London',
      address: 'London, UK',
      color: 'blue'
    }
  ];

  onMapClick(event: MapClickEvent) {
    console.log('Map clicked:', event.location);
  }
}

Template-based Popups

<ng-template #customPopup let-location let-pin="pin" let-deletePin="deletePin">
  <div>
    <h3>{{ pin.title }}</h3>
    <p>Lat: {{ location.latitude }}, Lng: {{ location.longitude }}</p>
    <button (click)="deletePin()">Delete Pin</button>
  </div>
</ng-template>

Advanced Features

Geocoding

The library includes built-in geocoding services using OpenStreetMap Nominatim API.

Search and Autocomplete

Integrated search functionality with autocomplete suggestions.

Selection System

Advanced selection system supporting both single and multi-select modes.

Multiple Tile Layers

Support for various map tile providers including OpenStreetMap, satellite imagery, and custom themes.
